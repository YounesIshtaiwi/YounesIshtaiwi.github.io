Keyboard Accessibility Instructions

This website is fully navigable using a keyboard. Here's how to access everything:

  • Press Tab to move forward through links and interactive elements.
  • Press Shift + Tab to move backward.
  • Press Enter to activate a selected link.
  • Use Alt + H (Windows) or Ctrl + Option + H (Mac) to go directly to the Home page.
  • Use Alt + T to go directly to the Things to Read page.
  • A “Skip to main content” link appears when using the keyboard, so you can bypass the navigation menu.

Keyboard Usability Principles Applied

  • Affordance: Navigation elements are styled to look and behave like links, giving cues for interaction.
  • Visibility: Focus indicators (highlight and border) clearly show which element is selected.
  • Feedback: Visual changes confirm the current focus point as users tab through the site.
  • Constraints: Only interactive elements are tabbable, simplifying navigation.
  • Mapping: Page layout matches typical web expectations, helping orientation.
  • Consistency: Same keyboard shortcuts and tab order are used across all pages.

This approach ensures keyboard users can fully navigate and interact with this website independently.
